## コマンド追加

headコマンドを追加。headコマンドは指定なしでは、最初の10行を表示する。10行は少し多いので7行にした。

`head -n 7`

確認のために表示するためのscriptを別に作成。

```
#!/bin/bash

cat -n $1 | sort -rn -k 2 | head -n 5

```
